# Prueba Técnica React - Login & Dashboard UI
Este proyecto es la solución a una prueba técnica en React. Su objetivo principal es crear una pantalla de login funcional, dando máxima prioridad a la presentación visual, la composición, los colores y la experiencia del usuario.

## Instrucciones de Ejecución
Para ejecutar este proyecto en tu entorno local, sigue estas instrucciones simples:

### Prerrequisitos
* Tener instalado **Node.js** en tu equipo.

### Instalación y Despliegue
1. Abre tu terminal o línea de comandos.
2. Navega hasta la carpeta raíz del proyecto.
3. Instala las dependencias necesarias ejecutando:
   ```bash
   npm install
   Una vez finalizada la instalación, levanta el servidor de desarrollo local ejecutando:
   npm run dev
   El terminal mostrará una dirección local (por defecto http://localhost:5173). Abre ese enlace en tu navegador para interactuar con la aplicación.
   Credenciales de acceso para evaluar la funcionalidad mínima y el acceso al dashboard , utiliza las siguientes credenciales quemadas:  
   Usuario: admin   
   Contraseña: 123456

### Características Implementadas
1. Tema Dinámico: 
    Se implementó un modo oscuro con selector de tema integrado.  
2. Responsive Design: 
    La interfaz se adapta perfectamente tanto a computadoras como a dispositivos celulares, garantizando una visualización en móvil y escritorio.  
3. Animaciones Sutiles: 
    Transiciones suaves, estados de carga y animaciones de entrada para mejorar la interacción.  
4. Accesibilidad (a11y): 
    Uso de etiquetas ARIA, buen contraste, navegación compatible con teclado y respeto por las preferencias de reducción de movimiento del sistema operativo.  
5. Dashboard Visual: 
    Se incluyó un pequeño panel de control estadístico posterior al inicio de sesión.  
6. Iconografía Elegante: 
    Uso de la librería lucide-react para complementar los textos y acciones de forma elegante.
